# 마스크 생성 가이드

이 문서는 도트 의상 생성용 마스크를 만들 때 기준을 흔들리지 않게 잡기 위한 안내서입니다.
기준 데이터 위치는 `E:\Aseet tool\data\clothing`입니다.

## 마스크의 역할

- `clothing_mask`는 옷이 새로 그려질 자리입니다.
- `preserve_mask`는 얼굴, 머리, 손발처럼 꼭 지켜야 할 부분입니다.
- 인페인팅에서는 흰색과 검은색을 잘 구분하는 것이 가장 중요합니다.

## 기본 규칙

1. 흰색은 작업할 영역입니다.
2. 검은색은 그대로 지킬 영역입니다.
3. 옷 마스크와 보존 마스크를 헷갈리면 결과가 맨몸처럼 나오거나 얼굴이 무너질 수 있습니다.
4. 마스크는 의상마다 따로 만드는 편이 더 안정적입니다.

## 어떤 부분을 칠해야 하나

### `clothing_mask`

- 상의라면 상체 옷이 들어갈 자리만 칠합니다.
- 원피스라면 상의와 하의를 함께 덮는 영역으로 칠합니다.
- 하의라면 허리 아래 옷 영역만 칠합니다.
- 옷이 들어갈 자리를 넉넉하게 잡되, 얼굴이나 손까지 크게 번지지 않게 합니다.

### `preserve_mask`

- 얼굴, 머리, 눈, 입, 손, 발처럼 꼭 남아야 하는 부분을 지킵니다.
- 옷이 겹치더라도 바꾸면 안 되는 부분은 보존 마스크로 막습니다.
- 보존해야 할 부분은 너무 작게 잡지 말고, 살짝 여유 있게 둘러줍니다.

## 만들 때 순서

1. 바디 기준 이미지를 엽니다.
2. 먼저 보존할 부분을 생각합니다.
3. 그다음 옷이 들어갈 영역을 칩니다.
4. 흰색과 검은색이 뒤집히지 않았는지 확인합니다.
5. 결과가 이상하면 마스크부터 다시 봅니다.

## 자주 생기는 실수

- 옷이 들어갈 자리를 검은색으로 두는 경우
- 얼굴이나 손까지 흰색으로 크게 칠하는 경우
- 의상 종류가 다른데 같은 마스크를 계속 쓰는 경우
- 정면용 마스크를 후면이나 다른 포즈에 그대로 쓰는 경우

## 확인 방법

- 결과가 옷이 아니라 맨몸처럼 나오면 마스크 방향이 잘못됐을 가능성이 큽니다.
- 얼굴이 무너지면 보존 마스크가 부족했을 가능성이 큽니다.
- 옷 형태가 너무 약하면 의상 마스크 범위를 조금 넓혀 봅니다.

## 추천 방식

- 먼저 `stand1` 같은 한 장으로 테스트합니다.
- 같은 의상은 기본 마스크를 하나 만들고, 필요할 때만 프레임별로 조금씩 손봅니다.
- 처음부터 124프레임 전체를 한 번에 맞추지 말고, 1장 성공 후 넓혀 갑니다.
- 실제 의상 파일은 `E:\Aseet tool\data\clothing` 아래에서 확인합니다.

## 의상이 많을 때 기준

- 의상이 260개 정도 있어도 전부 새로 만들 필요는 없습니다.
- 같은 실루엣끼리 묶어서 기본 마스크를 재사용합니다.
- 예를 들면 `상의형`, `원피스형`, `하의형`, `소매 큰 옷형`, `망토/외투형`처럼 나눌 수 있습니다.
- 먼저 몇 개의 기준 마스크를 만들고, 모양이 정말 다른 의상만 따로 수정하는 방식이 가장 현실적입니다.
- 전부를 1개씩 수작업하는 방식은 너무 오래 걸리고, 작업도 쉽게 지칩니다.
