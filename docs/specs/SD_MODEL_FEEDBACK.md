# 🧸 Dot Asset Tool: SD 1.5 vs SDXL 모델 선정 비교 보고서

메이플스토리 Worlds(MSW) 특유의 **2등신/3등신 SD 도트(Pixel Art) 의상 자산**을 생성하기 위해, **Stable Diffusion 1.5(SD 1.5)**와 **Stable Diffusion XL(SDXL)** 모델 중 어떤 환경이 더 적합한지에 대한 상세 기술 분석 리포트입니다.

---

## 1. 종합 결론: SD 1.5가 절대적으로 유리합니다.

메이플 풍 SD 도트의 핵심은 **극소 해상도(개별 43x68px, 그리드 120x120px)에서의 칼 같은 외곽선(Aliasing) 및 깔끔한 단색 명암**입니다. 종합적인 기술 분석 결과, **SD 1.5**를 메인 모델로 선정하는 것이 완성도와 제작 속도 면에서 압도적으로 우수합니다.

---

## 2. 세부 기술 비교 분석

### 2.1. 해상도 적합성 및 픽셀 묘사력 (Resolution & Pixel Aliasing)
* **SD 1.5 (512x512 최적화) ⭐ [매우 적합]**
  - SD 1.5는 512x512 해상도에서 이미지를 채우도록 설계되어 있습니다. 우리의 개별 프레임이나 가로 2~5개 프레임 그리드(예: 480x120px)를 처리할 때, 크기 차이가 크지 않아 **불필요한 디테일을 묘사하려 하지 않고 투박하면서도 귀여운 도트 느낌을 아주 잘 유지**합니다.
  - 도트 특유의 칼같이 끊어지는 경계면(Hard Edge)을 살리는 픽셀 LoRA와의 궁합이 탁월합니다.
* **SDXL (1024x1024 최적화) ❌ [부적합]**
  - SDXL은 1024x1024 해상도의 고화질, 초정밀 실사/일러스트 스타일을 그리도록 강하게 훈련되어 있습니다. 
  - 이를 도트 생성에 사용하면, 아주 작은 영역 내에서 **과도하게 부드러운 그라데이션(Anti-Aliasing)이나 수많은 미세 노이즈 도트를 그려버려 도트 본연의 '레트로한 맛'과 '투명 배경 후처리 시 외곽선 깔끔함'이 완전히 망가집니다.**

### 2.2. 도트 전용 LoRA 자원의 다양성 (LoRA Ecosystem)
* **SD 1.5 ⭐ [매우 풍부]**
  - Civitai 등 모델 허브에 공유된 **'Pixel Art / Retro Game Style'**, **'MapleStory Style / MSW Style'** 등 검증된 고품질 도트 전용 LoRA 자원의 90% 이상이 SD 1.5 기반으로 제작되어 있습니다.
  - 이 LoRA들은 아바타 바디 외곽에 1~2px 굵기의 단색 아웃라인(Black Outline)을 정교하게 고정해 주는 데 탁월한 성능을 보입니다.
* **SDXL ⚠️ [부족]**
  - SDXL 기반 도트 LoRA는 대다수 고해상도 Isometric 뷰의 3D 픽셀 게임 배경이나, 도트 일러스트레이션 풍에 치우쳐 있습니다. 메이플 같은 극단적 크기의 아바타 단일 의상 시트를 생성하는 데 맞는 전용 LoRA는 찾기 힘듭니다.

### 2.3. Google Colab 무료 환경 리소스 효율성 (Hardware Cost)
* **SD 1.5 ⭐ [매우 가볍고 빠름]**
  - 모델 크기가 약 2~4GB 내외로 가벼우며, Google Colab 무료 티어(T4 GPU 등)에서도 수 초 내에 인페인팅 1장이 생성됩니다.
  - **STEP 8 스타일 LoRA 학습** 시에도 학습 시간이 평균 30분~1시간 내외로 짧으며, VRAM 부족(Out of Memory)으로 인한 런타임 강제 종료 위험이 없습니다.
* **SDXL ❌ [무겁고 잦은 끊김]**
  - 모델 용량이 최소 6~12GB 이상이며, 추론 속도가 느립니다.
  - Colab 무료 버전에서 SDXL LoRA 학습은 메모리 제약 때문에 거의 불가능하거나, 유료 Colab(A100/V100 GPU) 결제가 불가피합니다.

## 3. 요약 비교표

| 비교 항목 | **Stable Diffusion 1.5** (추천) | **Stable Diffusion XL** |
| :--- | :--- | :--- |
| **권장 해상도** | 512 x 512 (우리 그리드 크기에 매우 근접) | 1024 x 1024 (너무 커서 도트 뭉개짐 발생) |
| **도트 외곽선 유지력** | 매우 우수 (Aliasing 보존 우수) | 나쁨 (Anti-Aliasing 필터로 인해 부드러워짐) |
| **도트 스타일 LoRA** | `Pixel Art`, `Maple` 등 다양하게 이미 존재 | 한정적이며 배경/일러스트 위주 |
| **Colab 구동 속도** | 1장 생성당 약 2~4초 | 1장 생성당 약 15~20초 |
| **LoRA 학습 비용** | 매우 저렴 (Colab 무료 환경에서 안정 구동) | 매우 비쌈 (OOM 오류 빈번, 유료 GPU 필수) |

---

## 4. 향후 추천 제작 공식 (Stable Diffusion 1.5 기반)

* **Base Model:** `v1-5-pruned-emaonly.safetensors` 또는 도트/애니 계열에 특화된 파인튜닝 모델 (예: `Anything V5`, `Counterfeit V3.0` 등)
* **Style LoRA:** `Pixel_Art_Style_LoRA` (가중치 `0.6` ~ `0.9`)
* **ControlNet:** `ControlNet-v1-1 Canny` (가중치 `0.4` ~ `0.7`로 포즈 윤곽선 고정)
* **Inpainting Setting:** Denoising Strength `0.35` ~ `0.55` 범위에서 그리드 시트 단위 인페인팅 수행.
